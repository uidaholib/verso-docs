## Display User CVs  
Seth Thompson 5/12/2026  

How to add a link to a CV on a Verso profile page

### Add the CV to the Web Server  
You will need access to the W: drive  
Convert the CV to PDF  
Save the CV to the following location in the web server W: drive:  

W:\www-lib-uidaho-edu\Content\media\verso\cvs\  

Once the PDF is saved in this location, double click it. The PDF should open in your web browser.  
Save this URL. Copy it or keep it open in the browser.  

### Adding the CV to a Verso Page  
Go to Esploro  
Set search bar to 'Researchers'  
Search for your researcher  
Ensure researcher type is 'Affiliated Researcher' (You can click the 'Affiliated' tab to refine your search)  
Click 'More Actions' on your desired researcher (three horizontal dots)  
Click 'Edit Profile'  
You are now 'logged in' as that user on Verso  
Click 'Edit Profile' above the profile picture  
Click 'About'  
In the Links section, click 'Add link'  
For title, enter 'CV'  
For URL, paste the browser URL from the PDF you saved to the web server earlier.  
Click 'Add'  
Close the Verso window so that you do not remain logged in as that user.  
Open Verso again to double check that the link is displayed and functional.  
Notify the researcher that you have successfully linked their CV. They are free to update the link title and description.
